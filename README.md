# TastyBlog — Современный блог-проект

## Описание проекта

**TastyBlog** — учебный проект фронтенд-разработки, демонстрирующий современные веб-технологии:
- Семантическая HTML5 разметка
- Адаптивная CSS3 верстка (Mobile First)
- Интерактив на чистом JavaScript ES6+
- Автоматическое форматирование Prettier
- Профессиональная структура файлов
- Интеграция с Figma (мудборд)
